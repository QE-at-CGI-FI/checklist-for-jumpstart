# The Enchanted Brew Shop

A fictional potion shop website created for the #28DaysOfTesting challenge.

## About

This website was built for participants of the 28 Days of Testing challenge to practice their exploratory testing skills.

## Challenge

- **Day 3**: Explore this website. No docs, no context. You have 15 minutes. What do you test first?
- **Day 5**: Pick your best bug. Write a proper bug report.

## How to Use

Visit: https://epictestquest.github.io/the-Potion-Shop/

Open the site. Explore. Test. Find things.

Good luck! 🧙‍♂️

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript

## Local Development

To run locally, simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000`
